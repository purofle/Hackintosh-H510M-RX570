# Hackintosh-H510M-RX570

## 版本

OpenCore: 0.9.9

macOS: 14.4.1

## 配置

|组件|配置|
|--|--|
|CPU|I9-11900K|
|内存|24G 2666Mhz 混搭 开启 XMP|
|显卡|蓝宝石 RX 570 8G|
|网卡|Intel AX200|
|SSD|移速 512GN2000|

- [x] USB端口
- [x] 蓝牙/WIFI
- [x] 声卡 HDMI/板载
- [x] Handoff(接力)/iMessage/FaceTime
